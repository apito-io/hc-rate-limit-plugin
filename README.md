# hc-rate-limit-plugin

API throttling and protection for the Apito API platform.

## Features

- Request rate limiting
- IP-based restrictions
- Custom time windows
- Burst protection
- Analytics dashboard

## Installation

```bash
go build -o hc-rate-limit-plugin .
```
